電源に関する推奨事項

1. 常に5V 3A以上の定格を持つ電源を使用してください。USBバッテリーを使用する場合、一部のUSBバッテリーの電源出力が弱いことに注意してください。
2. 十分な電流定格のUSBケーブルを使用してください。付属のUSBケーブルの電流定格は5Aです。

これら二つの点が、動作の遅さ、極端に短いWIFI範囲、不安定な接続問題の原因となることがよくあります。
